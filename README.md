# pc-huatihui-journal

## 项目简介

`pc-huatihui-journal` 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面通常以主题或专题形式组织，不针对特定网站或域名，主要用于记录、展示或分享与某些话题相关的静态内容。

本仓库旨在提供一个简洁、可直接访问的 HTML 页面集合，适合作为个人或小团队的资料存档与展示空间。

## 目录说明

```
pc-huatihui-journal/
├── index.html          # 主入口页面，可浏览所有归档页面
├── pages/              # 存放各独立 HTML 页面
│   └── ...             # 按日期或主题命名的 HTML 文件
├── assets/             # 页面所需的资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `index.html`：仓库首页，提供页面列表或导航。
- `pages/`：所有独立 HTML 页面均放置在此目录下，建议按 `YYYY-MM-DD-title.html` 或 `topic-name.html` 格式命名。
- `assets/`：存放页面共同依赖的样式、脚本和图片资源，便于统一维护。

## 页面归档说明

本仓库中的每个 HTML 页面均为独立可运行的静态文件，无需后端支持。页面内容可能涵盖各类主题，但不会指向任何具体商业网站或服务。

归档方式：
- 每个页面代表一个独立的记录或展示单元。
- 页面之间无强耦合关系，可单独访问。
- 资源文件统一存放在 `assets/` 目录，页面通过相对路径引用。

## 维护说明

- 新增页面：将 HTML 文件放入 `pages/` 目录，并在 `index.html` 中添加对应链接。
- 资源更新：如需修改公共样式或脚本，请更新 `assets/` 下对应文件，并确保各页面引用路径正确。
- 删除页面：移除 `pages/` 中对应文件，并同步更新 `index.html` 中的链接。
- 本仓库不计划提供评论、搜索或动态功能，保持纯静态结构。

## 许可

本仓库内容仅供学习与归档使用，不涉及商业用途。如需引用或修改，请保留出处说明。
